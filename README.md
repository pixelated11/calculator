# calculator
A simple calculator program written in C++. 

# Downloading Binary
I have compiled the binary my own, in release 0.1 with the executable. You don't need to compile the code, if you want it, you could get the .cpp file and compile it yourself using GCC. You can find the releases attached with a binary [here](https://github.com/pixelated11/calculator/releases).

# Compiling Yourself
If you want to compile the code, download the .cpp file yourself, then place it in any directory. Open a terminal/command prompt in the directory where the.cpp file lives, then. type in <br>
```g++ <filename> -static -static-libgcc -static-libstdc++ -o calculator.exe```
<br>
After you've done that, the executable named calculator.exe should be there. You can run it and enjoy calculating!
**Note:** <filename> is the .cpp file name.
